Dataset: MNIST handwritten Digits dataset. 
Dataset has been downloaded and converted to .csv file  using code provide by (http://ufldl.stanford.edu/wiki/index.php/Using_the_MNIST_Dataset) website. 
To run KNN for single test image click on KNN.m file and run it. 
Provide the value of K, length of train set and test image number in the sequence. 

The find the overall accuracy use Scoring.m file. Use GPU to run this file. Without GPU it will take 48+ hours to run it on nomal CPU with 8GB ram.
